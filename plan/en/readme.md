## 1.句子成分

### 1.1 句子成分构成

```
    主语、谓语、宾语、表语、补语、定语、状语。
```

#### 主语

```
    一句话叙述的主体。

    什么样的词可以充当主语?
        名词、代词、数词、动词不定式、动名称、句子、其他。
    （1）名词:
     Nick is a teacher (nick是人名(名词),作主语)
    The book is expensive (book 普通名词作主语)
    English is easy (专有名词 English 作主语)
    (2) 人称代词/指示代词:
    代词: 避免重复,代替名词
```
